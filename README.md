# Kamare (kamare.koplugin)

**kamare** is a manga reader plugin for [KOReader](https://github.com/koreader/koreader) that 
connects to [Kavita](https://github.com/Kareadita/Kavita).

This KOReader plugin provides a seamless way to read manga from Kavita without going through OPDS, 
featuring a custom renderer that also supports long-strip formats commonly used in manhwa.

This plugin offers a direct way to read your collection without the hassle of going through OPDS. 
Kamare features a custom renderer that is optimized for a fluid manga reading experience.

## Features

- Connects to your Kavita server to fetch manga titles and metadata.
- Optimized renderer for long-strip manga formats.
- Intuitive navigation between chapters and pages.
- Fast page turns through prefetching and caching.
- Integrates with KOReader reading stats.

Tested with Kindle Paperwhite (2nd gen) and Kobo Libra Colour.

## Installation & Usage

1. Download the latest `kamare.koplugin.zip` from the [releases page](https://github.com/faparicior/kamare.koplugin/releases/latest).
2. Extract the ZIP and copy the `kamare.koplugin` folder into the `plugins` directory of your KOReader installation.
3. Restart KOReader to load the plugin.
4. Open the plugin from the KOReader menu and configure it with your Kavita server details.
5. Browse and read your manga collection directly from Kavita.
